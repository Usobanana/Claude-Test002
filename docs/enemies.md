# エネミー仕様

## エネミー種別

| 種別 | 概要 |
|---|---|
| ザコ（雑魚） | 道中に配置される基本エネミー。倒しながらボスエリアへ進む |
| エリートザコ | ザコよりHP・攻撃力が高い中間的な強敵 |
| ボス | フィールドの最深部に配置される主要目標 |

---

## 属性

5属性が存在し、エネミーにも属性が設定される。

| 属性 | 備考 |
|---|---|
| 火 | 初期実装属性 |
| 水 | 今後追加 |
| 風 | 今後追加 |
| 光 | 今後追加 |
| 闇 | 今後追加 |

> まずは**火属性**のエネミーから実装を開始する

---

## エネミー共通ステータス

プレイヤーと同様のステータス構造を持つ（拡張性のため）。

| ステータス | 説明 |
|---|---|
| HP | 最大体力 |
| 攻撃力 | 与えるダメージの基礎値 |
| 防御力 | 受けるダメージを軽減 |
| 属性耐性 | 各属性に対する耐性値（%） |

---

## 種別ごとの設計方針

### ザコ
- HPは低め、攻撃力も控えめ
- 道中をテンポよく突破できるよう調整
- 状態異常を付与できる種類も作る（連携スキルのトリガー源として）

### エリートザコ
- ザコの2〜3倍程度のHPと攻撃力を想定
- 固有の行動パターンを持たせる
- ドロップ報酬が良い

### ボス
- フィールドの主要目標
- 独自のフェーズ・行動パターンを設計
- 属性ごとに個別のボスを作成予定
