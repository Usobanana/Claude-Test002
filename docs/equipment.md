# 装備品仕様（防具・装飾品）

## 設計方針

原神のアーティファクトシステムを参考にした構成。
将来的に変更・拡張することを想定し、まず同様の構造で実装する。

---

## 防具

### スロット構成

| スロット | メインパラメーター |
|---|---|
| 頭 | HP（固定） |
| 胴 | 攻撃力（固定） |
| 手 | 防御力% / HP% / 攻撃力%（ランダム） |
| 足 | 攻撃力% / HP% / 防御力%（ランダム） |

> 手・足のメインパラメーターは入手時にランダムで決定される

### 強化レベル

- **Lv1〜Lv15**
- 一定レベルごとにサブパラメーターが1枠上昇 or 新規追加（原神に準拠）
- 強化間隔の詳細（例：+3ごとに1回）は要調整

### サブパラメーター

- **最大4枠**
- 入手時にランダムで1〜4枠付与（レアリティによって初期数が異なることを想定）
- 強化のたびに既存のサブが上昇するか、空き枠に新規サブが追加される

#### サブパラメーター候補一覧

| ステータス |
|---|
| HP |
| HP% |
| 攻撃力 |
| 攻撃力% |
| 防御力 |
| 防御力% |
| 会心率 |
| 会心ダメージ |
| スタミナ |
| スタミナ% |

> 候補の追加・削除は要検討

---

## 装飾品

### スロット構成

- **2箇所**（ピアス・指輪など種類は問わない自由枠）

| スロット | メインパラメーター |
|---|---|
| 装飾品1 | 会心率 / 会心ダメージ / 攻撃力%（ランダム） |
| 装飾品2 | 会心率 / 会心ダメージ / 攻撃力%（ランダム） |

> メインパラメーターの候補は防具の手・足と同様に調整可能

### 強化・サブパラメーター

防具と同じシステムを適用（Lv1〜15、サブ最大4枠）

---

## 強化・合成

- 鍛冶屋で強化を実施
- 強化素材の詳細は要検討（専用素材 or 同種防具の合成など）

---

## 今後の検討事項

- 手・足・装飾品のメインパラメーター候補の最終決定
- サブパラメーター候補の追加・調整
- 強化レベルアップのタイミング（何Lvごとにサブが増えるか）
- レアリティシステム（初期サブ数・メインパラメーターの幅に影響）
- 強化素材の種類
