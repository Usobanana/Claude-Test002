# ゲーム概要

## 基本情報

| 項目 | 内容 |
|---|---|
| ジャンル | マルチプレイ アクションRPG |
| テーマ | ファンタジー |
| プラットフォーム | PC、スマートフォン |
| 視点 | トップダウンビュー |

## コンセプト

モンスターハンターのように**拠点とフィールドを行き来しながら、ボスを倒すことを主目的**としたゲーム。

## ゲームループ

```
拠点
 ├─ 装備の作成・強化
 ├─ パーティー編成
 └─ フィールドへ出発
      ↓
フィールド
 ├─ スタート地点からボスエリアまで移動
 ├─ 道中のザコエネミーを倒して進む
 └─ ボス戦（比較的短い道のり）
      ↓
クリア → 報酬獲得 → 拠点に帰還
```
