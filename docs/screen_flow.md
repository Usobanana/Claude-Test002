# 画面構成・遷移

## 大きな画面遷移

```
タイトル画面
    ↓
  拠点
    ↓（ギルドでクエスト受理 → 出口からフィールドへ）
  フィールド
    ↓（ボス討伐）
  リザルト
    ↓
  拠点
```

## 画面一覧（大区分）

| 画面 | 概要 |
|---|---|
| タイトル画面 | ゲーム起動時の最初の画面 |
| 拠点 | 準備・強化・クエスト受理を行うハブ |
| フィールド | ザコ討伐〜ボス戦を行うメインゲームプレイ |
| リザルト | クエスト終了後の結果表示・報酬確認 |

> 各大区分の中に詳細画面（メニュー、鍛冶屋UI、スキル確認など）が入る想定  
> 詳細な画面・UIは別途検討

---

## マルチプレイ時の遷移

```
自分の拠点
    ↓（メニューからリーダーの拠点を選択）
リーダーの拠点（パーティー全員が集まる）
    ↓（リーダーがクエスト受注 → HUDに表示）
全員が拠点出口で「出発」を押す
    ↓
フィールド（全員で同時入場）
    ↓
リザルト → リーダーの拠点に戻る（全員）
```
