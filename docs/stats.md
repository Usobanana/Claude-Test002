# ステータス・ダメージ計算

## 設計方針

- 拡張性を重視した設計
- ダメージ計算はゼンレスゾーンゼロ（ZZZ）に近い乗算式を採用
- Lvアップ時の効果は保留（システムは先行実装）

---

## 基礎ステータス一覧

| ステータス | 説明 |
|---|---|
| HP | 最大体力。0になると戦闘不能 |
| 攻撃力 | 与えるダメージの基礎値 |
| スタミナ | 回避・ダッシュのコストとなるリソース。時間で自動回復する |
| 防御力 | 受けるダメージを軽減する |
| 会心率 | クリティカルヒットが発生する確率（0〜100%） |
| 会心ダメージ | クリティカルヒット時のダメージ倍率 |

**スタミナの仕様：**
- 回避・ダッシュを行うと消費する
- 時間経過で自動回復
- ダメージ計算とは無関係（防御力とは別物）

> 消費量・回復速度・最大値の数値は別途調整

---

## レベル・経験値システム

- バトルで経験値（EXP）を獲得してレベルアップ
- レベルアップ時の効果・ステータス成長については**保留**
- システム上はレベルの概念を先行実装しておく

---

## ダメージ計算式

ZZZの計算式を参考に、乗算構造を採用する。

### 基本式

```
最終ダメージ = 攻撃力 × スキル倍率 × ダメージボーナス補正 × 会心補正 × 防御補正 × 属性耐性補正
```

### 各要素の詳細

#### 攻撃力
```
実効攻撃力 = 基礎攻撃力 × (1 + 攻撃力%バフ合計) + 攻撃力加算バフ合計
```

#### スキル倍率
- 各スキルに固有の倍率を設定
- 通常攻撃、アクティブスキル、ウルトスキルそれぞれに個別倍率

#### ダメージボーナス補正
- 武器強化（+値）、バフなどによる追加ダメージ倍率
- 複数のボーナスは**加算**してから乗算
```
ダメージボーナス補正 = 1 + ダメージ増加率合計
```

#### 会心補正
```
会心非発生時 : 1.0
会心発生時   : 1 + 会心ダメージ率
期待値       : 1 + 会心率 × 会心ダメージ率
```

#### 防御補正
```
有効防御力 = 防御力 × (1 - 防御ダウン率) - 防御貫通値
防御補正   = プレイヤーLv係数 / (プレイヤーLv係数 + 有効防御力)
```
> Lv係数は後述の「レベル係数テーブル」で定義（要別途設計）

#### 属性耐性補正
```
属性耐性補正 = 1 - 敵の属性耐性値 + 属性ダメージ増加率
```
- 弱点属性攻撃時は耐性補正がプラスに働く（有利倍率）
- 複数の耐性補正は加算してから適用

## 拡張ポイント（将来対応）

- ブレイク/スタッガーシステム（敵がブレイク状態で追加倍率など）
- 元素反応・属性相性システムの拡張
- パーティーバフ・デバフの加算ロジック
- レベル係数テーブルの設計
