# 武器システム

## 概要

各キャラクターが使える武器の種類は固定。武器は拠点でクラフトおよび強化できる。

## キャラクター別 武器種

| キャラクター | 使用可能武器 |
|---|---|
| 戦士 | 片手剣 |
| 騎士 | 片手剣、盾 |
| 射手 | 弓 |

## クラフト

- 拠点で素材を使って武器を作成できる
- 素材はフィールドでのドロップなどを想定（詳細は別途）

## 強化システム

- 同じ武器同士を合成することで強化値（+値）が上昇する
- 強化の上限は **+10**

| 強化値 | 内容 |
|---|---|
| +0 | 初期状態 |
| +1〜+9 | 強化中 |
| +10 | 最大強化 |

> 合成によって+値を積み上げていく方式（例：+3の武器 + +3の武器 → +4に近づくなど、詳細ルールは要検討）
